##  Getting Started

### Prerequisites
- Node.js (v14 or above)
- npm or yarn

### Installation & Running Locally

```bash
# Clone the repository
git clone https://github.com/praveenudayagiri/devTinder.git

# Enter the project directory
cd devTinder

# Install dependencies
npm install
# or
yarn install

# Start development server
npm run dev
# or
yarn dev
